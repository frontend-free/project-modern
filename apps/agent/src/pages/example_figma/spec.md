# 华润三九智能体广场 — 功能与技术方案

> 设计稿：Figma [华润三九智能体广场](https://www.figma.com/design/XR8UZSBfydB69MOPk1W6yK/?node-id=66-8542)

## 功能描述

智能体广场作为「智能体市场/目录」页面，包含：顶部精选轮播、我收藏的智能体、全部/我的作品切换、分类筛选、搜索、智能体卡片列表（含收藏/分享等交互）。

---

## 页面结构（自上而下）

| 区域                | 功能说明                                                                                                                |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **顶部导航**        | 标题「华润三九智能体广场」+ 右侧主按钮「注册AI智能体创作者」                                                            |

## 功能清单与实现建议

### 3.1 顶部导航

- **标题**：华润三九智能体广场（文案可配置）
- **主按钮**：注册AI智能体创作者（跳转或弹窗，按产品约定）
- **技术**：使用 @fe-free/core 的 PageLayout 头部，或自定义 Header + Antd `Button` type="primary"

### 3.2 精选轮播 Banner

- **行为**：左右箭头切换，展示多个「精选智能体」位（设计稿为 2 个：医药研发智能体、营销智能体）
- **单屏内容**：背景图、标题、描述文案(2行截断)
- **技术**：使用 **Antd Carousel**，箭头为自定义左右箭头

### 3.3 我收藏的智能体

- **有数据**：展示用户收藏的智能体（可与下方卡片复用同一卡片组件，仅数据源为「收藏列表」）
- **空态**：标题 + 「您还没有收藏任何智能体」+ 「浏览智能体并点击⭐️图标进行收藏」+ 占位图标（如星形）
- **技术**：条件渲染；空态可用 Antd `Empty` 或自定义占位区块

### 3.4 Tab 切换

- **Tab 项**：「全部」「我的作品」
- **技术**：使用 **@fe-free/core 的 Tabs 组件**，与项目约定一致

### 3.5 分类筛选

- **选项**：所有分类、工作助手、文本创作、商业助手、制造业、金融、医疗健康、科研、教育
- **表现**：选中态（如「所有分类」）与未选中态样式区分（设计稿为圆角 Tag，选中为蓝底白字）
- **技术**：使用 **Antd Tag** 或 Tag 组横向排列，点击切换筛选条件，与列表联动

### 3.6 搜索

- **占位**：「搜索智能体」
- **行为**：输入后按产品约定做实时搜索或回车搜索，结果驱动下方智能体列表
- **技术**：Antd `Input` / `Input.Search`，或 ProComponents 搜索组件，样式参考设计稿（圆角、边框）

### 3.7 智能体卡片列表

- **布局**：网格，每行 2 张卡片（可响应式）
- **单卡片**：图标(左)、标题、描述（多行截断）、星标数/点赞数/评论数、右侧分类 Tag（多个）、分享图标；支持「收藏」切换（星标描边/填充）
- **技术**：使用 AgentItem组件

### 3.8 其他交互（可选在 spec 中标注）

- 收藏/取消收藏：点击卡片星标，更新「我收藏的智能体」与列表态
- 分享：点击分享图标，按产品约定（复制链接/调起分享面板等）
- 卡片点击：进入智能体详情或使用页（路由与产品约定一致）

## 技术栈与规范（与项目一致）

- **UI**：Antd、@fe-free/core、@fe-free/icons
- **样式**：Tailwind，设计变量参考 `@fe-free/core/src/tailwind.css`，避免硬编码色值
- **路由**：约定式路由，跳转用 `useNavigate` 或 `routeTool`
- **目录与命名**：小写+下划线；组件使用 `export { XXX }`，页面入口可用 `export default`
